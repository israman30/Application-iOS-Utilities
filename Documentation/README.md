## Documentation

This folder contains **usage snippets + instructions** for each component in this repository.

### How to use these docs

- **If you consume the framework**: add `Utils/Utils.xcodeproj` to your app, then `import Utils`.
- **If you copy source files**: copy the relevant file(s) from `Utils/Utils/` into your target.
- **To see the sample display**: open `Utils/Utils.xcodeproj` → open the component file under `Utils/Utils/` → run the SwiftUI preview (`*_Previews` / `*SampleView`).
